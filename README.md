# Bank-Account-Management-System-Python-

This project is a console-based banking application developed in Python that simulates core banking operations using object-oriented programming principles. It features two main classes: BankAccount (encapsulating account details and transactions) and BankSystem (managing multiple accounts and providing a user-friendly menu interface).

Key functionalities include:

Account creation with validation for duplicate accounts and non‑negative initial balance.

Deposit and withdrawal operations with error handling for negative amounts and insufficient funds.

Balance inquiry to retrieve current account balance.

Money transfer between accounts with rollback logic in case of failure.

Robust input validation and clear error messages to guide user interaction.

The project demonstrates proficiency in Python syntax, class design, encapsulation, dictionary data structures, and procedural user interaction. It highlights the ability to build a functional, modular system with real‑world applicability—ideal for showcasing foundational programming skills and understanding of banking logic.
